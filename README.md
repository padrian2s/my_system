# my_system

TUI file manager with time-sorted listing built with Textual.

## Overview

A terminal user interface (TUI) file manager that displays files sorted by modification time. Built using Python's Textual framework for a rich terminal experience.

## Contents

- `lstime.py` -- Main application script
- `lstime/` -- Package module
- `setup.sh` -- Setup/installation script

## Getting Started

```bash
bash setup.sh
python lstime.py
```

Requires Python 3 and the Textual library.
